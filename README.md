【概要】
Yostar提供スマートフォン向けアプリゲーム『ブルーアーカイブ』のキャラクター育成支援ツール。

①キャラクターを選択
②育成状況について現在・目標の数値をそれぞれ入力
以上２ステップで、目標までに集める必要のある素材の数を一括計算


【仕様】
AWS EC2(Amazon Linux2)+Nginx+uWSGI+Django
