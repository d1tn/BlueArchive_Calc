# 概要
Yostar提供のスマートフォン向けアプリゲーム『ブルーアーカイブ』のキャラクター育成支援ツール。

# 使い方
- ①キャラクターを選択
- ②育成状況について現在・目標の数値をそれぞれ入力
上記２ステップで、目標までに集める必要のある素材の数を一括計算

# 構成
AWS EC2(Amazon Linux2)+Nginx+uWSGI+Django
